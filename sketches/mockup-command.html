Capabilities
The tools you need, built into your shell

AI Chat

Ask anything about commands, debugging, or automation. Works with any OpenAI-compatible API.

Smart SSH

SSH by server name, not IP. Fuzzy matching, multi-hop via --via. YAML inventory.

SSH Escape

AI REPL during SSH. Ask questions, run !commands, no remote agent needed.

Workflows

YAML-based multi-step automation: SSH, SCP, local commands — all in one file.

RTK Compress

Auto-compress terminal output 60-95% before AI processing. Zero config.

Pipe Mode

cmd | opsterm "analyze" — pipe any output for instant AI analysis.

SCP Transfer

Upload/download files by server name: opsterm scp file.txt srv:/path.

Completion

Bash + Zsh auto-complete for servers, workflows, and subcommands.

History

SQLite-backed history. Revisit, explain, or re-run past commands with AI.

Quick Install

  1. 1 Download release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Place in ~/.local/bin
  4. 4 Add AI provider key
Single Python file
Any AI provider
Multi-hop SSH
RTK compression
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
