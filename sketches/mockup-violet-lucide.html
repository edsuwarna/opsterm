Capabilities
Everything you need, right in your shell

AI Chat

Ask anything — commands, explanations, debugging. Works with any OpenAI-compatible provider.

Smart SSH

SSH by server name, not IP. Fuzzy matching, multi-hop via --via, YAML config.

SSH Escape

AI REPL inside SSH sessions. Ask questions, run !commands, no server-side install.

Workflows

Multi-step YAML automation: SSH → SCP → local commands in a single file. Repeatable.

RTK Compress

Auto-compress output 60-95% with RTK before AI. Zero config, works out of the box.

Pipe Mode

cmd | opsterm "analyze" — explain output, detect errors, get actionable insights.

SCP Transfer

Upload/download via server name: opsterm scp file.txt server:/path.

Tab Completion

Bash + Zsh auto-complete for servers, workflows, and subcommands. Fast.

History

SQLite-backed command history. Revisit, explain, or re-run past commands.

Quick Install

  1. 1 Download latest release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Install to ~/.local/bin
  4. 4 Configure your AI provider
Single Python file
Any AI provider
Multi-hop SSH
RTK compression
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
