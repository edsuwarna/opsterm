OpsTerm is a local AI assistant that lives in your terminal. SSH into any server without leaving your AI behind — because the AI runs on YOUR machine, not the remote server.

~/opsterm — zsh ● ● ●
# Ask AI for anything
$ opsterm how to find large log files
$ find /var/log -type f -size +100M -exec ls -lh {} \;

# SSH by name, not IP
$ opsterm ssh vps-utama

ubuntu@203.0.113.10:~$

# Explain any output
$ docker ps | opsterm "any errors?"

$ opsterm connect vps-utama

AI CHAT

Ask anything — commands, explanations, scripts. Any OpenAI-compatible provider.

SMART SSH

SSH by server name. Fuzzy matching, multi-hop via --via.

SSH ESCAPE

AI REPL during SSH. Ask questions, run !commands, no server install.

Getting Started
ONE COMMAND

QUICK INSTALL

  1. 1 Download latest release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Install to ~/.local/bin
  4. 4 Configure AI provider
✓ SINGLE PYTHON FILE
✓ ANY AI PROVIDER
✓ MULTI-HOP SSH
✓ RTK COMPRESSION
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
