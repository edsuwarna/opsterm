Capabilities
Built for engineers who work in the terminal

AI Chat

Ask anything about your commands, errors, and scripts. Any OpenAI-compatible provider, zero config.

Smart SSH

SSH by server name, not IP. Fuzzy matching, multi-hop --via, YAML inventory.

SSH Escape

AI REPL during SSH sessions. Ask questions, !commands, zero remote install needed.

Workflows

YAML-based multi-step automation: SSH, SCP, local commands — one repeatable file.

RTK Compress

Auto-compress terminal output 60-95% before AI sends it. Zero config.

Pipe Mode

cmd | opsterm "analyze" — pipe any output and get instant AI insights.

SCP Transfer

Upload/download by server name: opsterm scp file.txt srv:/path.

Completion

Bash + Zsh auto-complete for servers, workflows, and subcommands. Fast.

History

SQLite-backed. Revisit, explain with AI, or re-run past commands.

Getting started
One curl, zero dependencies

Quick Install

  1. 1 Download latest release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Place in ~/.local/bin
  4. 4 Add your AI provider key
Single binary
Any AI provider
Multi-hop SSH
RTK compression
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
