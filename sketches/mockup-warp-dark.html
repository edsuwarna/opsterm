Your terminal, augmented by AI

OpsTerm is a local AI terminal assistant. SSH into any server without leaving your AI behind — because the AI runs on your machine, not the remote.

~/opsterm — zsh
# Ask AI for anything
$ opsterm how to find large log files
$ find /var/log -type f -size +100M -exec ls -lh {} \;

# SSH by name, not IP
$ opsterm ssh vps-utama
🔑 Connecting to vps-utama…
ubuntu@203.0.113.10:~$

# Explain any output
$ docker ps | opsterm "any errors?"

# SSH escape — AI mid-session
$ opsterm connect vps-utama

Capabilities
Everything you need in the terminal
🤖

AI Chat

Ask anything — commands, explanations, scripts. Works with any OpenAI-compatible provider.

🔑

Smart SSH

SSH by server name, not IP. Fuzzy matching, multi-hop via --via, YAML config.

🔮

SSH Escape

AI REPL during SSH. Ask questions, run !commands, no server-side install.

⚡

Workflows

Multi-step YAML automation: SSH → SCP → Local commands in a single file.

🗜️

RTK AI

Auto-compress output 60-95% with RTK before AI. Zero config, works out of the box.

🔗

Pipe Mode

cmd | opsterm "analyze" — explain output, detect errors, get insights.

📁

SCP Transfer

Upload/download via server name: opsterm scp file.txt server:/path

⌨️

Tab Completion

Bash + Zsh auto-complete for servers, workflows, and subcommands.

📋

History

SQLite-backed history. Revisit, explain, or re-run past commands.

Getting started
One command, zero dependencies

Quick Install

  1. 1 Download latest release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Install to ~/.local/bin
  4. 4 Configure your AI provider
✓ Single Python file
✓ Any AI provider
✓ Multi-hop SSH
✓ RTK compression
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
