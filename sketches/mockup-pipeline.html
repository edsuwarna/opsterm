Capabilities
Everything you need, at your fingertips

AI Chat

Ask anything — commands, explanations, debugging. Any OpenAI-compatible provider, zero config.

Smart SSH

SSH by server name, not IP. Fuzzy matching, multi-hop via --via, YAML inventory.

SSH Escape

AI REPL inside SSH. Ask questions, run !commands, zero server-side install required.

Workflows

Multi-step YAML automation: SSH → SCP → local commands. Repeatable, versionable, one file.

RTK Compress

Shrink output 60-95% with RTK before AI analysis. Zero config, zero overhead, out of the box.

Pipe Mode

cmd | opsterm "analyze" — pipe output directly, get insights, detect errors instantly.

SCP Transfer

Upload and download via server name: opsterm scp file.txt srv:/path. Simple.

Completion

Bash + Zsh auto-complete for servers, workflows, and subcommands. Blazing fast.

History

SQLite-backed command history. Revisit, explain with AI, or re-run past commands.

Getting started
One command, zero dependencies

Quick Install

  1. 1 Download release via curl
  2. 2 Export OPSTERM_VER
  3. 3 Place in ~/.local/bin
  4. 4 Add your AI provider key
Single Python file
Any AI provider
Multi-hop SSH
RTK compression
# Install latest release
$ curl -L https://raw.githubusercontent.com/
edsuwarna/opsterm/main/LATEST \
-o /tmp/opsterm_latest
$ export OPSTERM_VER=$(cat /tmp/opsterm_latest)
$ curl -L "https://raw.githubusercontent.com/edsuwarna/opsterm/${OPSTERM_VER}/bin/opsterm" \
-o ~/.local/bin/opsterm
$ chmod +x ~/.local/bin/opsterm
$ opsterm provider add default --api-key 'sk-...'
