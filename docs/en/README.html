рҹ§  Design Philosophy

  1. Zero dependencies вҖ” just Python 3 stdlib, no pip install needed (except vault encryption)
  2. Single file вҖ” bin/ai can be copied to any server and run immediately
  3. Local-first вҖ” AI runs on your local terminal, not on the server. SSH without losing AI access
  4. Config as code вҖ” server & workflow configs use YAML, version-control friendly
  5. Progressive disclosure вҖ” simple features are easy to use, complex features are available when needed
